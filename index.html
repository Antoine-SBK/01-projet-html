Balise de base

<h> = (1 a 6, du plus grand au plus petit)
<p> = paragraphe
<b> = texte en gras
<i> = texte en italique
<u> = texte souligné
<a href= url=""/a> = lien de base
<sup> = Plus petit texte placé au-dessous du texte normal
<sub> = Plus petit texte placé au-dessous du texte normal
<mark> = surligne le mont selectionner
<br /> = Aller a la ligne (dans un paragraphe)
<ul> <li> </li> </ul> = permet de faire des listes

<a href target ="_blank"> = attribue qui permet d'ouvrir une nouvelle page quand clique sur plusieur lien
<a href="#(nom)"> = permet de rester sur la page tout en ciblant un lien
<hr / > = permet du "dessiner" un train qui separe la page
<address > = permet de specifier l'adresse du web master
<del > = ercit le mots et l'affiche barré a l'ecrant
<time > = permet d'inserer une date dans texte
<pre > = permet de prendre en compte les espaces dans le texte
<sup > = permet d'incrémenter un nombre (le décaler du mot vers le haut)
<sub > = permet d'incrémenter un nombre (le decaler du mot vers le bas: ex avec : H20 )

<em > = groupe de mot plus important (pour la recherche du moteur de recherche)
<strong > = groupe de mot plus important (plus important que em )
<cite > = specifie qu'il sagis d'une citation d'un auteur (nom de l'auteur)
<q > = inclure une citation courte dans le texte
<blockquote><p> = mettre une citation (utiliser surtour pour les citation longues) peut contenir des attribus comme lang= cite)
<img src="image.." alt=".." >=attribus pour les mal voyant"
<figure > = regroupement de fichier comme images
<figcaption >= donne un titre au image (le met au dessus)
